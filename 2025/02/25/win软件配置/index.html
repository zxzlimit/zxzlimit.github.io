win软件配置

发表于 2025-02-25 更新于 2025-04-15 分类于 软件 阅读次数： 本文字数： 987 阅读时长 ≈ 1 分钟
我很喜欢捣鼓一些电脑的配置，尤其喜欢搞一下软件来提升自己的效率，其实对于mac应为有时间备份基本不用在重新配置了，但是对于win，我找了很多备份的，但都没有能完全还原配置的。这里我就把我常用的软件罗列出来吧，比较都是经常用的，还有之前找这些东西也花了我不少时间的。

win软件配置

软件和插件

1、Typora

这是一个md文件的编辑软件经常使用的，感觉还不错吧

2、TTime

可以orc进行识别，效果还可以的

3、idm

宇宙级下载器，也不多说了，已经入了永久版了

4、Twinkle Tray

能调剂电脑的显示亮度，感觉还是很有用的

5、QuickLook

按空格快速浏览文件，类似于mac上的空格

6、TranslucentTB

一个能改变win底部框的小软件

7、Advanced Archive Password Recovery

暴力破解压缩包的工具

8、Clash for Windows

翻墙软件

9、PicList

图床软件，和typora相关一起用

10、QQ截图独立版

截图软件，好用

11、平板串流

这两个软件，第一个是虚拟屏幕，第二个是串流，详细配置去b站搜索平板串流有教程

12、Zoomlt

可以放大缩小屏幕，在屏幕上涂鸦，适合教学

13、火绒

真心不错的杀毒软件，它的弹窗拦截好评

14、Adobe Acrobat DC

pdf神奇了属于

15、draw.io

流程图神器

16、Quicker

绝对的神器，能快速启动软件，能自己编一些程序，然后通过其一键打开。十分十分有用对于我来说**********，当然这个配置也最麻烦。

17、篡改猴

油猴脚本，可以自己写一些js代码在浏览器上执行，很不错

18、沉汉式翻译

浏览器翻译的神

19、OBS Studio

录屏直播软件好评

20、Todesk

远程控制软件，也没什么好的，没办法暂时就这个好点

21、GIT

git的终端操作器好用

22、Docker

容器

23、Mysql

数据库

24、Xshell

远程连接服务器终端的

25、Postman

后端api测试工具

26、VMware

虚拟机

27、Hbuilder

前端的ide

28、Vscode

全能好用的ide

29、Intell

java的ide

30、Navicat Premium

远程数据库管理软件

31、Xmind

不多说，思维导图软件

32、figma

网页设计软件，不太会

33、WSL

win下的linux系统

34、Anaconda Navigator

python必装的东西

35、OneDrive

云盘

36、word实用工具插件

可以支持正则表达

37、方方格子和uuoffice

基本上就这些了，其他也没什么好说的了

38、LocalSend

局域网文件传输工具

39、SyncClipboard

云粘贴版

40、Everything

文件搜索神器
